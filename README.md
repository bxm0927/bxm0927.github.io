## Quick Start

```bash
# Install Hexo
$ npm install hexo-cli -g

# Setup your blog
$ hexo init blog
$ cd blog

# Start the server
$ hexo server

# Create a new post
$ hexo new "Hello Hexo"

# Generate static files
$ hexo generate
```
